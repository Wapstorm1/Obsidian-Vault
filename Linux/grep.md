[[]]- -i makes the search case-insensitive
    
- -n shows line numbers for matches
    
- -c counts how many matching lines exist
    
- -v shows lines that do not match
    
- -w matches whole words only
    
- -r searches through directories recursively
    
- -o shows only the part of the line that matched
    
- -E enables extended regular expressions
    
- --color highlights matched text in output
    
- -l shows only filenames that contain matches
    

  

These flags can be combined. You now know how to search with precision.

## **PASS 1 — Static Visual Scan (No Interaction)**

  

Look for:

  

1️⃣ Alignment

- Elements centered properly
    
- Margins consistent
    
- No overlapping
    

  

2️⃣ Hierarchy

- Is primary action clearly dominant?
    
- Is price more visible than secondary text?
    

  

3️⃣ Text

- Overflow
    
- Truncation
    
- Wrong font size
    
- Currency formatting
    

  

4️⃣ Visibility

- Any hidden or partially visible elements?
    
- Button not fully clickable?
    

---

## **PASS 2 — Interaction Scan**

  

Now tap everything:

- Confirm Purchase
    
- Terms link
    
- Close button
    
- Back gesture
    

  

Check:

- Button disabled state
    
- Loading state
    
- Double tap behavior
    
- Rapid tap behavior
    
- Keyboard overlapping fields
    

---

## **PASS 3 — Stress & Edge**

- Rotate device
    
- Background → return
    
- Long text (large price)
    
- Large accessibility font
    
- Dark mode
    
- One-handed reachability

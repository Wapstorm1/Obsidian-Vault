**XML (eXtensible Markup Language)**

• **What it is**:
• A markup language that uses a tree structure for representing data.
• Data is encapsulated in user-defined tags.

• **Example**:

```
<user>
    <id>1</id>
    <name>Alice</name>
    <email>alice@example.com</email>
    <isActive>true</isActive>
</user>
```
